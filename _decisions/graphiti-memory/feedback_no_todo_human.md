项目开发session中禁止使用 Learn by Doing / TODO(human) 模式让用户写代码。

**Why:** 用户明确反馈"我们当前是项目开发，又不是练习"。Learn by Doing 适合教学场景，不适合实际项目推进。用户是甲方，AI 是执行者。

**How to apply:** 在 Canvas Learning System 项目的所有开发 session 中，直接实现代码，不插入 TODO(human) 占位符。Insight 教学注释可以保留（用户未反对），但不要中断工作流让用户写代码。
