# Canvas Learning System v2.0 - Master Control

## Metadata
- **Command**: /v2
- **Description**: Master control center for Canvas Learning System v2.0 features
- **Bmad Pattern**: Core system control with keyword activation
- **Version**: 2.0
- **Dependencies**: canvas_utils.py, loguru

## Usage

### Basic Activation
```bash
/v2          # Show system status dashboard
/v2 *ultrathink    # Toggle UltraThink intelligent strategy system
/v2 *review        # Toggle Ebbinghaus review system
/v2 *concurrent    # Toggle multi-agent concurrent processing
/v2 *graph         # Toggle Graphiti knowledge graph
/v2 *monitor       # Toggle enterprise error monitoring
```

### Advanced Configuration
```bash
/v2 config         # Show current configuration
/v2 config review  # Configure Ebbinghaus review system
/v2 config agents  # Configure agent behavior
/v2 reset         # Reset all v2.0 features to default
```

## Implementation

基于Bmad设计模式的核心控制系统，支持`*`关键词激活和全局状态管理。

**Global Features Status**:
- UltraThink System: `/v2 *ultrathink`
- Ebbinghaus Review: `/v2 *review`
- Concurrent Processing: `/v2 *concurrent`
- Knowledge Graph: `/v2 *graph`
- Error Monitoring: `/v2 *monitor`

**Automatic Features** (Always On):
- Canvas file backup system
- Basic error logging
- Agent coordination framework
- Learning session tracking

使用此命令作为所有v2.0功能的中央控制台。
