# 🎯 对比表Agent - 结构化对比易混淆概念

**这个Agent是干什么的？**

对比表Agent专门处理"容易混淆"的概念。它会生成结构化的对比表格，从定义、特征、使用场景、示例、常见错误等多个维度对比2-4个相似概念，帮助你清晰区分。

**核心能力**：结构化对比，区分易混淆概念

**输出格式**：生成 `.md` 文件（Markdown表格）

---

## 📝 什么时候用这个Agent？

### ✅ 适用场景

- **概念易混淆**：多个概念长得很像，容易搞混
- **不知道区别**：知道每个概念，但说不清区别
- **选择困难**：做题时不知道该用哪个概念
- **对比学习**：想通过对比加深理解
- **典型例子**：
  - 逆命题 vs 否命题 vs 逆否命题
  - 充分条件 vs 必要条件
  - 归纳法 vs 演绎法

### ❌ 不适用场景

- **单个概念不懂**：如果是理解单个概念，用 `/澄清路径` 或 `/口语化解释`
- **需要深度理解**：如果需要1500+词深度讲解，用 `/澄清路径`
- **记忆问题**：如果区别懂了但记不住，用 `/记忆锚点`

## ⚙️ Agent工作原理

### 输入要求

- **概念列表**：2-4个需要对比的概念
- **上下文**：可选 - Canvas中的相关内容

### 输出内容：多维度对比表

生成的对比表包含以下维度：

#### 1. 定义对比
```
| 概念 | 定义 |
|------|------|
| 逆命题 | 交换原命题的条件和结论：q→p |
| 逆否命题 | 否定后交换：¬q→¬p |
```

#### 2. 特征对比
```
| 概念 | 核心特征 | 与原命题关系 |
|------|---------|-------------|
| 逆命题 | 交换位置 | 不等价 |
| 逆否命题 | 否定+交换 | 等价 |
```

#### 3. 使用场景对比
```
| 概念 | 何时使用 | 典型场景 |
|------|---------|---------|
| 逆命题 | 反向思考 | 探索充分必要性 |
| 逆否命题 | 反证法 | 否定形式更简单时 |
```

#### 4. 示例对比
```
| 概念 | 示例 |
|------|------|
| 逆命题 | 原:下雨→地湿  逆:地湿→下雨 ❌ |
| 逆否命题 | 原:下雨→地湿  逆否:地不湿→没下雨 ✅ |
```

#### 5. 常见混淆点
```
| 易错点 | 说明 |
|--------|------|
| 混淆逆命题和逆否命题 | 记住：只有逆否等价！ |
| 忘记否定 | 逆否=先否定再交换 |
```

### 生成的文件

- **文件名**：`对比表-{概念A}-vs-{概念B}-{时间戳}.md`
- **位置**：Canvas同目录
- **格式**：Markdown表格，便于打印和查阅

## 📊 对比表示例

### 示例：逆命题 vs 否命题 vs 逆否命题

**生成的对比表**：

```markdown
# 对比表：逆命题 vs 否命题 vs 逆否命题

## 1. 定义对比

| 类型 | 符号表示 | 操作 | 与原命题关系 |
|------|---------|------|-------------|
| **原命题** | p → q | - | 本身 |
| **逆命题** | q → p | 交换条件和结论 | ❌ 不等价 |
| **否命题** | ¬p → ¬q | 否定条件和结论 | ❌ 不等价 |
| **逆否命题** | ¬q → ¬p | 先否定再交换 | ✅ 等价 |

## 2. 生活例子对比

| 类型 | 例子 | 是否成立 |
|------|------|---------|
| 原命题 | 下雨 → 地湿 | ✅ 成立 |
| 逆命题 | 地湿 → 下雨 | ❌ 不成立（可能是洒水） |
| 否命题 | 不下雨 → 地不湿 | ❌ 不成立（可能洒水） |
| 逆否命题 | 地不湿 → 没下雨 | ✅ 成立（等价） |

## 3. 真值表对比

| p | q | p→q | q→p | ¬p→¬q | ¬q→¬p |
|---|---|-----|-----|-------|-------|
| T | T | T   | T   | T     | T     |
| T | F | F   | T   | T     | F     |
| F | T | T   | F   | F     | T     |
| F | F | T   | T   | T     | T     |

结论：第1列和第4列完全相同 → 原命题 ≡ 逆否命题 ✅

## 4. 使用场景对比

| 类型 | 何时使用 | 为什么用 |
|------|---------|---------|
| 逆命题 | 探索充要性 | 检验是否充要条件 |
| 否命题 | 很少用 | 与原命题不等价，意义不大 |
| 逆否命题 | 反证法、间接证明 | 等价且否定形式可能更简单 |

## 5. 记忆口诀

| 类型 | 口诀 |
|------|------|
| 逆命题 | "反着说" - 只交换 |
| 否命题 | "全否定" - 只否定 |
| 逆否命题 | "否完再反" - 先否定再交换 |

**关键**：只有"否完再反"才等价！

## 6. 常见错误

| 误解 | 正确理解 |
|------|---------|
| ❌ 逆命题也等价 | ✅ 只有逆否命题等价 |
| ❌ 否定就是加"不"| ✅ 否定要正确使用逻辑符号¬ |
| ❌ 三者都差不多 | ✅ 三者完全不同，只有逆否等价 |
```

**阅读对比表后填写理解**：88分 → 变绿色 ✅

## 🆘 常见问题

**Q: 对比表和澄清路径有什么区别？**
A:
- **澄清路径**：深度解释单个概念（1500+词）
- **对比表**：结构化对比多个概念（表格形式）

**Q: 可以对比几个概念？**
A: 建议2-4个。太多会导致表格混乱，太少不需要对比。

**Q: 对比表能对比不同类型的概念吗？**
A: 最好对比"同类但易混淆"的概念。完全不同类型的概念没有对比价值。

**Q: 对比表看完还是记不住怎么办？**
A: 使用 `/记忆锚点` 生成记忆口诀和生动类比。

## 🎓 最佳实践

### ✅ 最适合对比的概念

- 名字相似（逆命题、否命题、逆否命题）
- 定义相似（充分、必要、充要）
- 容易选择困难（什么时候用哪个？）
- 学生经常混淆的

### ❌ 不适合对比的

- 完全不同类型（苹果 vs 汽车）
- 只有一个概念不懂（先理解再对比）
- 概念数量>5个（太复杂）
