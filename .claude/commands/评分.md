# 🎯 评分Agent - 4维度智能评分（准确性+形象性+完整性+原创性）

**这个Agent是干什么的？**

评分Agent是Canvas学习系统的核心评估工具。它对你在黄色节点中填写的个人理解进行4维度评分（每维25分，总分100），根据分数自动流转节点颜色（红→紫→绿），并智能推荐后续Agent。

**核心能力**：量化评估理解质量，驱动学习循环

**评分哲学**："词可达意"即可 - 用自己的话说清楚就算理解

---

## 📝 什么时候用这个Agent？

### ✅ 适用场景

- **填写完黄色节点**：你已经用自己的话解释了概念
- **检验理解质量**：想知道自己理解得怎么样
- **颜色流转**：让系统判断是红色/紫色/绿色
- **获取反馈**：想知道哪个维度需要改进
- **UltraThink工作流**：评分后自动生成follow-up节点

### ❌ 不适用场景

- **还没填写理解**：黄色节点为空时无法评分
- **不是理解节点**：只能评分黄色节点，不能评分问题节点

## ⚙️ Agent工作原理

### 输入要求

- **问题内容**：问题节点的内容（红色/紫色）
- **用户理解**：黄色节点中你填写的个人理解（**必需**）
- **参考材料**：可选 - 原始材料用于对比准确性

### 4维度评分标准（每维25分，总分100）

#### 1. 准确性 (Accuracy) - 25分
```
评分标准：
- 23-25分：完全准确，核心正确
- 20-22分：基本准确，有小瑕疵
- 15-19分：部分准确，有明显错误
- 0-14分：理解错误，偏离核心

关键：不要求100%精确，但核心必须对
```

#### 2. 形象性 (Imagery) - 25分
```
评分标准：
- 23-25分：有生动例子/类比，易于想象
- 20-22分：有例子但不够生动
- 15-19分：只有抽象描述，无例子
- 0-14分：完全抽象，难以理解

关键：能否用具体例子说明抽象概念
```

#### 3. 完整性 (Completeness) - 25分
```
评分标准：
- 23-25分：覆盖核心要点，结构完整
- 20-22分：覆盖主要要点，略有遗漏
- 15-19分：只说了一部分，遗漏较多
- 0-14分：严重不完整，遗漏核心

关键：是否说全了？核心要点都提到了吗？
```

#### 4. 原创性 (Originality) - 25分
```
评分标准：
- 23-25分：用自己的话，有个人理解
- 20-22分：基本是自己的语言
- 15-19分：部分照搬教材/资料
- 0-14分：完全复制粘贴

关键：是自己的理解还是抄的？
```

### 颜色流转规则

```
总分 ≥ 80  → 🟢 绿色（完全理解，通过）
总分 60-79 → 🟣 紫色（似懂非懂，待深化）
总分 < 60  → 🔴 红色（不理解，待学习）
```

### 智能推荐（Story 2.9创新）

根据弱项维度推荐Agent：

| 弱项维度 | 推荐Agent | 原因 |
|---------|----------|------|
| Accuracy低 | `/澄清路径` `/口语化解释` | 需要正确理解概念 |
| Imagery低 | `/记忆锚点` `/对比表` | 需要具象化理解 |
| Completeness低 | `/澄清路径` `/四层次解释` | 需要全面理解 |
| Originality低 | `/口语化解释` `/记忆锚点` | 需要用自己的话 |

## 📊 评分报告示例

### 示例1：高分（85分）- 绿色

**你的理解**：
```
逆否命题就是把原命题的条件和结论都否定后，再交换位置。
它和原命题是逻辑等价的，所以可以用来证明原命题。

举个例子："如果下雨，地面就湿"的逆否命题是
"如果地面不湿，就没下雨"，意思是一样的。

我理解逆否命题的用处是：当正面证明难的时候，
可以证明逆否命题（往往否定形式更简单）。
```

**评分报告**：
```
总分：85分 🟢 绿色

维度分数：
- Accuracy（准确性）：24分 ✅
  核心理解正确：否定+交换，等价性理解准确

- Imagery（形象性）：22分 ✅
  有生动例子（下雨和地湿），易于理解

- Completeness（完整性）：21分 ✅
  覆盖定义、等价性、应用场景，结构完整

- Originality（原创性）：18分 ✅
  用自己的话，有个人理解（"往往否定形式更简单"）

反馈：
很好！你已经理解了逆否命题的核心。
建议：可以补充一下为什么等价（真值表或逻辑推理）

颜色变更：红色 → 🟢 绿色
```

### 示例2：中等（65分）- 紫色

**你的理解**：
```
逆否命题就是把原命题反过来说。
和原命题意思一样，所以可以互换。
```

**评分报告**：
```
总分：65分 🟣 紫色

维度分数：
- Accuracy（准确性）：16分 ⚠️
  "反过来说"不准确，应该是"否定+交换"

- Imagery（形象性）：15分 ⚠️
  完全没有例子，过于抽象

- Completeness（完整性）：18分 ⚠️
  只说了定义和等价性，遗漏应用场景

- Originality（原创性）：16分 ⚠️
  语言过于简略，看不出深度理解

反馈：
你理解了基本概念，但不够深入。

改进建议：
1. 准确性：说清楚"反过来"具体指什么（否定+交换）
2. 形象性：加一个生活例子
3. 完整性：补充什么时候用逆否命题证明

智能推荐Agent：
- 提高准确性 → /澄清路径
- 增加形象性 → /记忆锚点
- 补充完整性 → /四层次解释（进阶层）

颜色变更：红色 → 🟣 紫色
```

### 示例3：低分（42分）- 保持红色

**你的理解**：
```
逆否命题就是命题的否定形式。
```

**评分报告**：
```
总分：42分 🔴 红色

维度分数：
- Accuracy（准确性）：8分 ❌
  理解错误！逆否≠否定，是"否定+交换"

- Imagery（形象性）：10分 ❌
  完全没有例子，一句话太抽象

- Completeness（完整性）：12分 ❌
  严重不完整，只有一句话

- Originality（原创性）：12分 ❌
  看起来是背的定义，没有真正理解

反馈：
你对逆否命题的理解有误。
逆否命题不等于否定形式！

关键区别：
- 否命题：只否定（¬p → ¬q）❌ 不等价
- 逆否命题：否定+交换（¬q → ¬p）✅ 等价

必须行动：
1. /澄清路径 或 /口语化解释 重新学习概念
2. 理解后重新填写黄色节点
3. 再次评分

智能推荐Agent：
- 首选：/澄清路径（深度系统化理解）
- 备选：/口语化解释（轻松理解）

颜色：保持 🔴 红色
```

## 🆘 常见问题

**Q: 评分标准太严格了吗？**
A: 不严格！评分哲学是"词可达意"即可：
- 不要求学术表达
- 不要求100%精确
- 只要核心正确、用自己的话说清楚就行

**Q: 为什么我理解了但分数只有60多？**
A: 可能的原因：
- Accuracy：有小错误或不够准确
- Imagery：缺少具体例子
- Completeness：遗漏了某些要点
- Originality：语言过于教材化

看评分报告的"反馈"部分，有具体改进建议！

**Q: 怎样才能拿高分（≥80）？**
A: 满足以下条件：
1. ✅ 核心理解准确（不要有明显错误）
2. ✅ 至少有1个具体例子
3. ✅ 覆盖主要要点（定义+特点+应用）
4. ✅ 用自己的话（不照抄教材）

**Q: 智能推荐的Agent一定要用吗？**
A: 不一定，这是建议：
- 如果你认同推荐，使用推荐的Agent
- 如果你有其他想法，可以自己选择
- 如果已经≥80分，不需要任何Agent

## 💡 提高评分的技巧

### 技巧1：提高Accuracy（准确性）

❌ 不准确的表达：
```
"逆否命题就是反过来说"
```

✅ 准确的表达：
```
"逆否命题是先否定条件和结论，再交换位置（¬q → ¬p）"
```

### 技巧2：提高Imagery（形象性）

❌ 纯抽象：
```
"逆否命题和原命题等价"
```

✅ 加例子：
```
"逆否命题和原命题等价。
比如'下雨→地湿'等价于'地不湿→没下雨'，意思完全一样。"
```

### 技巧3：提高Completeness（完整性）

问自己3个问题，确保完整：
1. ✅ 定义说了吗？（是什么）
2. ✅ 特点说了吗？（有什么特点，如等价性）
3. ✅ 应用说了吗？（什么时候用）

### 技巧4：提高Originality（原创性）

❌ 照抄教材：
```
"设原命题为p→q，则逆否命题为¬q→¬p，
且(p→q) ⟺ (¬q→¬p)"
```

✅ 用自己的话：
```
"我理解逆否命题就是：
原来说'有A就有B'，现在说'没B就没A'，
其实是一个意思，只是换个角度说。"
```
