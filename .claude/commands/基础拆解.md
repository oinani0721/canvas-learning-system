# 🎯 基础拆解Agent - 将看不懂的材料拆解为引导性问题

**这个Agent是干什么的？**

基础拆解Agent专门处理"完全看不懂"的难懂材料（红色节点）。它会将抽象、复杂的内容拆解为3-7个简单、循序渐进的引导性问题，帮助你从"大脑宕机"状态过渡到"似懂非懂"状态。

**核心能力**：降低认知门槛，建立基础理解

---

## 📝 什么时候用这个Agent？

### ✅ 适用场景

- **完全不懂**：看到材料大脑一片空白
- **太抽象**：概念太抽象，没有具体印象
- **红色节点**：标记为红色的完全不理解的内容
- **建立认识**：需要从零开始建立基础认识

### ❌ 不适用场景

- **似懂非懂**：如果你已经有一些理解，应该用 `/深度拆解`
- **需要记忆**：如果理解了但记不住，应该用 `/记忆锚点`
- **需要解释**：如果需要系统化解释，应该用 `/澄清路径`

## ⚙️ Agent工作原理

### 输入要求

- **材料内容**：完全不懂的材料文本
- **主题名称**：核心概念的名称
- **用户理解**：可选，如果有初步理解可以提供

### 输出内容

生成3-7个引导性问题，包含4种类型：

1. **定义型**：最简单的定义是什么？
2. **实例型**：有没有生活中的例子？
3. **对比型**：和你熟悉的X概念有什么区别？
4. **探索型**：为什么需要这个概念？

每个问题都带有💡提示，降低思考难度。

### 处理流程

```
输入难懂材料
    ↓
识别核心概念和困难点
    ↓
拆解为3-7个循序渐进的问题
    ↓
在Canvas创建紫色问题节点
    ↓
你填写黄色理解节点
```

## 🔗 配合使用建议

### 推荐工作流

1. **第一步**：`/基础拆解` 红色节点
2. **第二步**：填写黄色理解节点
3. **第三步**：`/评分` 黄色节点
4. **第四步**：
   - 如果评分<60 → 继续使用 `/澄清路径` 或 `/口语化解释`
   - 如果评分60-79 → 使用 `/深度拆解`
   - 如果评分≥80 → 节点变绿，继续下一个

### 与UltraThink智能策略系统配合

- **智能v1生成**：系统自动检测理解程度0-30%时，会自动使用基础拆解策略
- **手动强制**：如果你想手动指定，使用本命令 `/基础拆解`
- **策略切换**：如果基础拆解后评分仍<50，系统会自动切换到补充解释策略

## 📊 与其他拆解Agent的区别

| Agent | 适用场景 | 输入要求 | 输出类型 |
|-------|---------|---------|---------|
| **基础拆解** | 完全不懂（红色） | 不需要理解 | 3-7个引导性问题 |
| **深度拆解** | 似懂非懂（紫色） | 需要已有理解 | 3-10个检验性问题 |
| **问题拆解** | 应用题求解 | 需要题目描述 | 突破性问题 |

## 🎓 实际案例

### 案例1：拆解"鸽笼原理"

**你的状态**：红色节点，完全不懂

**拆解后的问题**：
1. （定义型）最简单来说，鸽笼原理是在说什么？
2. （实例型）生活中有什么例子可以体现这个原理？
3. （探索型）为什么这么简单的道理还要起个名字？
4. （对比型）它和"抽屉原理"是一回事吗？

**填写理解后评分**：50分 → 转紫色 → 继续深度拆解

## 🆘 常见问题

**Q: 为什么拆解后还是不懂？**
A: 基础拆解只是降低门槛，你还需要填写理解+补充解释。推荐配合 `/澄清路径` 使用。

**Q: 拆解的问题太简单了？**
A: 说明你不是"完全不懂"，应该用 `/深度拆解` 而不是基础拆解。

**Q: 我想自己指定问题怎么办？**
A: 你可以手动在Canvas创建紫色节点，不一定要用Agent生成。
