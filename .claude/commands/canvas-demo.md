## 📚 示例1：学习离散数学中的"逆否命题"

### 场景描述
学生完全不理解"逆否命题"这个概念。

### 完整流程

**Step 1: 在Obsidian中创建Canvas**
```
文件名：笔记库/离散数学/离散数学.canvas
节点内容："逆否命题是什么？它和原命题有什么关系？"
节点颜色：红色（完全不理解）
```

**Step 2: 基础拆解（Claude Code中）**
```
帮我基础拆解 @笔记库/离散数学/离散数学.canvas 中的"逆否命题"节点
```

**Agent输出**:
- 生成3-7个引导性问题（定义型、实例型、对比型）
- 为每个问题创建黄色理解节点
- 自动布局在Canvas上

**Step 3: 填写个人理解（Obsidian中）**
```
在Obsidian中打开Canvas文件
点击黄色节点
用自己的话填写对每个问题的理解
保存文件（Ctrl+S）
```

**Step 4: 评分（Claude Code中）**
```
帮我评分 @笔记库/离散数学/离散数学.canvas 中所有黄色节点
```

**Agent输出**:
- 4维度评分（准确性、具象性、完整性、原创性）
- 总分：例如 72分
- 颜色流转：黄色 → 紫色（60-79分，似懂非懂）
- 智能推荐：推荐使用 clarification-path 和 oral-explanation

**Step 5: 补充AI解释（Claude Code中）**
```
帮我生成口语化解释 @笔记库/离散数学/离散数学.canvas 中的"逆否命题"
```

**Agent输出**:
- 创建文件：`逆否命题-oral-explanation-20250115123045.md`
- 内容：800-1200词教授式解释
- 在Canvas上创建蓝色节点链接到该文件

**Step 6: 优化理解后再评分（Claude Code中）**
```
帮我评分 @笔记库/离散数学/离散数学.canvas 中所有黄色节点
```

**Agent输出**:
- 总分：例如 85分
- 颜色流转：紫色 → 绿色（≥80分，完全理解）

## 🎨 生动类比

**原命题**就像说："如果下雨（P），我就带伞（Q）"

**逆否命题**就是："如果我没带伞（¬Q），说明没下雨（¬P）"

想象自己是侦探，看到没带伞的人 → 推断没下雨

## 🎵 记忆口诀

**"前后都取反，逻辑等价关"**

前 = P → Q
后 = ¬Q → ¬P
取反 = 否定
等价 = 真值完全相同
...

直接证明较难 → 使用逆否命题

## 💡 最佳实践建议

### 原白板学习
1. **从易到难**: 先用basic-decomposition，再用deep-decomposition
2. **强制输出**: 必须填写黄色节点，不能跳过
3. **及时评分**: 填写后立即评分，获得反馈
4. **针对性补充**: 根据评分维度选择解释Agent

### 检验白板学习
1. **真实检验**: 不看原白板，暴露盲区
2. **动态扩展**: 检验白板上继续拆解和解释
3. **持续迭代**: 直到80%绿色再停止
4. **对比分析**: 检验白板 vs 原白板，找差距

### 常见错误
❌ 只看不写（没有填写黄色节点）
❌ 直接看AI解释（跳过自我输出）
❌ 评分<60分就放弃（应该继续拆解）
❌ 检验白板看原白板（失去检验意义）
