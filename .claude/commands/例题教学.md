# 🎯 例题教学Agent - 完整的解题教程（~1000词）

**这个Agent是干什么的？**

例题教学Agent通过完整的例题解答来教学。它生成约1000词的详细教程，包含：题目、思路分析、分步求解、易错点提醒、变式练习和答案提示。适合通过"做题学概念"的学习方式。

**核心能力**：例题驱动教学，在解题中理解概念

**输出格式**：生成 `.md` 文件（~1000词，6个section）

**关键特点**：Learn by Doing - 在实际应用中学习

---

## 📝 什么时候用这个Agent？

### ✅ 适用场景

- **会概念不会做题**：理解了定义，但不知道怎么用
- **需要看例题**：通过例题学习比看定义更有效
- **问题拆解后仍不会**：用了 `/问题拆解` 后还是做不出来
- **需要练习**：想看完整的解题步骤和思路
- **学习解题方法**：想掌握某种题型的解题套路

### ❌ 不适用场景

- **概念完全不懂**：如果概念都不理解，应该先用 `/澄清路径` 或 `/口语化解释`
- **只是不会某一道题**：如果只是某道题不会，用 `/问题拆解` 更合适
- **理论理解需求**：如果需要深度理论讲解，用 `/澄清路径`

## ⚙️ Agent工作原理

### 输入要求

- **概念/题型**：必填 - 需要例题的概念或题型
- **难度要求**：可选 - 简单/中等/困难
- **具体题目**：可选 - 如果有具体题目可以提供

### 输出内容：6个Section（~1000词）

#### 📝 Section 1：题目（100-150词）
```
- 完整的题目描述
- 已知条件列举
- 求证/求解目标
- 背景说明（如果需要）
```

#### 💡 Section 2：思路分析（150-200词）
```
- 这道题考什么？
- 关键信息是什么？
- 应该用什么方法？
- 为什么用这个方法？
- 解题的突破口在哪？
```

#### 🔢 Section 3：分步求解（300-400词）
```
Step 1: [第一步]
详细说明每一步在做什么、为什么这样做

Step 2: [第二步]
...

Step 3: [第三步]
...

最终答案：...
```

#### ⚠️ Section 4：易错点提醒（100-150词）
```
❌ 常见错误1：...
✅ 正确做法：...

❌ 常见错误2：...
✅ 正确做法：...
```

#### 🔄 Section 5：变式练习（150-200词）
```
变式1（稍微改变条件）：...
变式2（改变题型但同样方法）：...
变式3（提高难度）：...

这些变式帮你巩固方法！
```

#### 💭 Section 6：答案提示（100词）
```
变式1提示：...
变式2提示：...
变式3提示：...

（不直接给答案，给思路提示）
```

### 生成的文件

- **文件名**：`例题教学-{概念}-{时间戳}.md`
- **位置**：Canvas同目录
- **长度**：~1000词（6个section）

## 🎓 实际案例

### 案例：例题教学 - "逆否命题证明"

**生成的教学文档**（节选）：

```markdown
# 例题教学：逆否命题证明

## 📝 题目

**题目**：证明以下命题

原命题：如果 n² 是偶数，则 n 是偶数（n ∈ ℕ）

**要求**：
1. 写出逆否命题
2. 用逆否命题进行证明
3. 说明为什么用逆否命题更简单

---

## 💡 思路分析

### 这道题考什么？
- 考查逆否命题的概念
- 考查逆否命题在证明中的应用
- 考查何时选择逆否证明

### 关键信息
- 原命题：n² 偶 → n 偶
- 要用逆否命题证明
- 涉及偶数和奇数的性质

### 为什么用逆否命题？
**直接证明**：从 n² 是偶数 → 推出 n 是偶数
  问题：n² 给的信息太少，不好直接推 n

**逆否证明**：从 n 是奇数 → 推出 n² 是奇数
  优势：奇数的性质更容易处理！

### 解题突破口
逆否命题：如果 n 是奇数，则 n² 是奇数
这个更容易证明！

---

## 🔢 分步求解

### Step 1：写出逆否命题

原命题：n² 偶 → n 偶
逆否命题：n 奇 → n² 奇

（先否定两边，再交换位置）

### Step 2：证明逆否命题

**证明**：设 n 是奇数

则 n 可以写成：n = 2k + 1（其中 k ∈ ℤ）

计算 n²：
n² = (2k + 1)²
   = 4k² + 4k + 1
   = 2(2k² + 2k) + 1
   = 2m + 1（其中 m = 2k² + 2k ∈ ℤ）

因此 n² 是奇数（形如2m+1）

### Step 3：利用等价性得出结论

因为逆否命题成立，且逆否命题与原命题等价

所以原命题成立：如果 n² 是偶数，则 n 是偶数 ✅

### 最终答案

逆否命题：如果 n 是奇数，则 n² 是奇数
证明：n = 2k+1 → n² = 2(2k²+2k)+1 = 奇数
结论：原命题成立（等价性）

---

## ⚠️ 易错点提醒

### ❌ 错误1：写错逆否命题
**错误写法**：逆否是"n² 不是偶数 → n 不是偶数"

**问题**：没有明确说"不是偶数=奇数"（实数包括小数！）

✅ **正确写法**：
- 题目中 n ∈ ℕ，所以不是偶数就是奇数
- 明确写：n 是奇数 → n² 是奇数

### ❌ 错误2：直接证明原命题
**错误做法**：从 n² 偶 → 想直接推 n 偶

**问题**：这很难！n² = 偶数 → n = ?（没有明显推导）

✅ **正确做法**：
- 用逆否命题证明
- 奇数的性质很清楚：n = 2k+1

### ❌ 错误3：忘记说明等价性
**错误**：证明了逆否命题就结束

**问题**：没有说明"因为等价，所以原命题成立"

✅ **正确做法**：
- 最后一定要说：逆否命题成立 + 等价性 → 原命题成立

---

## 🔄 变式练习

### 变式1（相同方法）
证明：如果 n³ 是偶数，则 n 是偶数

### 变式2（相同思路，不同概念）
证明：如果 n² > 0，则 n ≠ 0
提示：考虑逆否命题

### 变式3（综合应用）
证明：如果 a + b 是无理数，则 a 或 b 至少有一个是无理数
提示：用逆否命题 + 反证法

---

## 💭 答案提示

### 变式1提示
- 逆否命题：n 奇 → n³ 奇
- 设 n = 2k+1，计算 n³
- n³ = (2k+1)³ = ... = 2m + 1（奇数）

### 变式2提示
- 逆否命题：n = 0 → n² = 0（不大于0）
- 这个逆否命题显然成立
- 因此原命题成立

### 变式3提示
- 逆否命题：a 和 b 都是有理数 → a+b 是有理数
- 有理数的和是有理数（容易证明）
- 因此原命题成立

---

## 📚 解题套路总结

### 何时使用逆否证明？
1. 直接证明"p → q"很困难
2. 否定形式"¬q → ¬p"更容易处理
3. 结论的否定有明确性质可用

### 逆否证明的步骤
1. 写出逆否命题（先否定，再交换）
2. 证明逆否命题
3. 说明等价性，得出原命题成立

### 常见应用场景
- 偶数/奇数相关证明
- 有理数/无理数证明
- 集合包含关系证明
- 逻辑推理题
```

## 🆘 常见问题

**Q: 例题教学和问题拆解有什么区别？**
A:
- **问题拆解**：生成3-5个引导问题，帮你找思路
- **例题教学**：完整解答+详细步骤，直接教你怎么做

**Q: 看完例题后应该做什么？**
A:
1. 合上文档，自己重新做一遍例题
2. 做文档中的变式练习
3. 把解题步骤总结成套路
4. 应用到新题目

**Q: 例题看懂了，但自己做不出怎么办？**
A: 说明你是"看懂"而不是"学会"。建议：
1. 不看文档，自己重新做
2. 卡住时再看对应的Step
3. 反复练习，内化为自己的方法

**Q: 可以为具体题目生成教学吗？**
A: 可以！提供具体题目，Agent会针对这道题生成完整教学。

## 💡 使用技巧

### 技巧1：主动做题，而不是看题

❌ 错误学习方式：
```
看完例题 → 觉得懂了 → 结束
```

✅ 正确学习方式：
```
看完例题 → 合上文档 → 自己做一遍 → 对照检查 → 做变式
```

### 技巧2：总结解题套路

每看完一道例题，总结：
- **识别特征**：什么特征的题用这个方法？
- **关键步骤**：核心步骤是什么？
- **易错点**：哪里最容易错？

### 技巧3：建立例题库

把例题教学文档分类保存：
```
逆否命题证明/
  ├── 偶数奇数类.md
  ├── 有理无理类.md
  └── 集合包含类.md
```

考前复习时快速翻阅。

### 技巧4：从易到难

```
1. 先看例题（中等难度）
2. 做变式1（简单）
3. 做变式2-3（提高）
4. 找更难的题练习
```
