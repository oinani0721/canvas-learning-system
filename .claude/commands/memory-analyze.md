# Canvas学习模式分析命令

## 功能描述

基于Story 8.17的学习活动记录，分析您的学习模式并生成个性化建议。

## 使用方式

```bash
/memory-analyze                          # 分析最近7天学习模式
/memory-analyze --days 30               # 分析最近30天
/memory-analyze --user-id user123      # 分析指定用户
/memory-analyze --style learning       # 专注学习风格分析
/memory-analyze --recommendations     # 仅显示建议
```

## 分析维度

### 🎯 学习风格分析 (8维度)
- **视觉学习偏好** - 图表、图像等视觉化内容
- **听觉学习偏好** - 语音解释、讨论等
- **动觉学习偏好** - 实践操作、具体应用
- **顺序学习倾向** - 循序渐进vs整体把握
- **抽象/具体倾向** - 概念理解vs实例应用
- **独立/协作倾向** - 个人学习vs合作学习
- **深度/广度倾向** - 深入钻研vs广泛涉猎
- **反思/行动倾向** - 思考总结vs立即实践

### 📊 行为模式分析
- **学习时段偏好** - 最佳学习时间
- **Agent选择模式** - 偏好的辅助工具
- **学习持续时间** - 单次学习长度
- **错误修正模式** - 从错误中学习的方式
- **知识迁移能力** - 跨领域应用能力

### 🤖 智能推荐
- **个性化Agent推荐** - 最适合您的学习工具
- **学习路径优化** - 基于历史数据的学习顺序
- **复习时机建议** - 基于遗忘曲线的复习提醒
- **难度调整建议** - 个性化难度推荐

## 数据要求

- 需要有至少3天的学习活动记录
- 建议有20+个学习活动以获得准确分析
- 分析准确性随数据量增加而提升
