## 使用技巧

### 如何自定义快捷键

1. 打开 设置 (Ctrl/Cmd + ,)
2. 点击左侧 "快捷键"
3. 搜索 "Canvas"
4. 点击命令旁的 "+" 按钮添加快捷键

### 快捷键冲突

如果快捷键与其他插件冲突，Obsidian会显示警告。
建议使用 `Mod+Shift` 组合以减少冲突。

### 跨平台说明

- `Mod` 在 macOS 上是 `⌘ (Command)`，在 Windows/Linux 上是 `Ctrl`
- `Alt` 在 macOS 上是 `⌥ (Option)`
