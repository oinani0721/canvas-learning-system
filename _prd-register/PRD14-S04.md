# PRD14-S4 · 6 个核心 Skill 设计

> **From PRD**: S4 [6 个核心 Skill 设计] (line 3604-4840)

## 核心内容

设计 6 个 Claude Code Skill 的 hotkey 绑定、workflow 和 MCP 调用链。包括 `/start_exam_board`（Cmd+Option+E，检验白板灵魂 skill）、`/chat_with_context`（Cmd+Option+C，带上下文的节点对话）、`/edge_discuss`（Cmd+Option+R，EI+SE 双策略 Edge 对话）、`/quiz_from_callout`（Cmd+Option+Q，批注驱动考察）、`/extract_node`（Cmd+Option+X，拉出新节点）、`/review_profile`（Cmd+Option+P，学习档案综述）。所有 hotkey 均为默认值可自定义，支持 Obsidian 层级和 Claude Code 层级两种修改方式。
