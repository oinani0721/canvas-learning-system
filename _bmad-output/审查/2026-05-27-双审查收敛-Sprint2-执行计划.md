# 双审查收敛 → Sprint 2 执行计划定稿

> ⛔ **这是你从"凌乱"到"清晰"的终点**: 两轮独立深度审查 (Claude 读代码实证 + ChatGPT 读设计+学习科学) 得出几乎相同的数字和**同一条主干诊断**。结论不再有分歧 → 可以定稿 Sprint 2 到底做什么。

## 1. 为什么这次结论可信 — 两轮独立审查收敛

| 维度 | Claude workflow (读代码) | ChatGPT DR (读设计+学习科学) | 收敛 |
|---|---|---|---|
| 原白板 | 真 68% | 5.6/10 (覆盖 55%) | ✅ "方向对, 未到生产力" |
| 检验白板 | 真 42% | 4.2/10 (覆盖 38%) | ✅ 高度一致 |
| 核心闭环 | 真 37.5% | 3.8/10 | ✅ 几乎相同 |
| 根因 | Graphiti 三管道全断 | 个人记忆脊柱未成形 | ✅ **同一诊断** |
| 唯一先手 | 接通脊柱 (非加功能) | 统一事件化记忆脊柱 | ✅ **同一处方** |

**两个独立方法得出同一结论 = 这不是猜测, 是真相。**

## 2. 唯一先手 — 个人记忆脊柱 (ChatGPT 定稿, Claude 实证支撑)

```
批注/关系/对话/校准
   → 统一事件化 (CanvasGraphEpisodeV1)
   → Graphiti 版本链 (belief_key) + LanceDB 精确片段
   → 真实题面持久化 (questions_registry)
   → relation-aware 出题 (wikilink 邻居进主路径)
   → 评分回读真实题面
   → 节点档案回写 (mastery/FSRS)
```

**在这条主干成形前, 不要优先做更多 UI 花样。** 原白板没有它只是"更聪明的编辑器", 检验白板没有它只是"会引用你一句话的问答器"。

## 3. Sprint 2 锁定 — 5 个接通任务 (零新功能蔓延)

|---|---|---|---|---:|---|

**Sprint 2 总工时 37h** (按历史 ×2 经验值约 60-74h, 多窗口并行约 2-4 天)。

### ⚠️ ChatGPT 新增 — 静默评分哲学修正 (并入 S2-4)

你锁定"完全静默不打断心流", 但 Claude 实证 `exam-quick.ts:146` 实际是保存后追加 `## 反馈 (4/5)` + 弹 Notice = **延迟显式评分, 违背你的哲学**。S2-4 顺手改成:
- 保存后只看**轻量状态标记** (如 ✓ 已评)
- 详细反馈**默认折叠** / 一轮后集中回看
- 分数先进**节点 profile**, 不占答题现场
