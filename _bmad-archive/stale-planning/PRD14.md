# PRD14: Canvas Learning System

Canvas Learning System is a Tauri 2 + Obsidian Hybrid desktop learning application built around active recall, spaced repetition, and a personal knowledge graph. The system enables learners to engage in context-aware AI dialogue, take assessments on a blank recall board, track mastery via BKT+FSRS fusion, and maintain a Graphiti-backed long-term memory.
