  # Canvas Learning System - Epic 12 用户指南

  ## 概述

  Epic 12 引入了**三层记忆系统**和**Agentic RAG**，显著提升了学习体验：

  - **检索质量提升**: MRR@10 从 0.25 提升到 0.38+
  - **检验题相关性**: 从 70% 提升到 85%+
  - **检索延迟**: P95 < 400ms

  ---

  ## 新功能介绍

  ### 1. 三层记忆系统

  #### Layer 1: Graphiti时序知识图谱
  - **作用**: 记录学习历程和概念关系
  - **特点**: 时间感知，自动追踪学习进度

  #### Layer 2: LanceDB向量数据库
  - **作用**: 高性能语义检索
  - **特点**: 多模态支持，P95延迟<50ms

  #### Layer 3: Temporal Memory
  - **作用**: 艾宾浩斯复习调度
  - **特点**: 基于FSRS算法，个性化复习间隔

  ### 2. Agentic RAG检索增强

  新的检索流程采用LangGraph编排：

  ```
  Query → 并行检索(Graphiti + LanceDB) → 融合 → Reranking → 质量评分 → 结果
  ```

  **融合策略**:
  - RRF: 通用场景
  - Weighted: 检验白板生成 (薄弱点权重70%)
  - Cascade: 高精度需求

  **Reranking策略**:
  - Cohere: 高质量场景 (检验白板)
  - Local: 日常检索 (零成本)

  ### 3. 检验白板生成改进

  使用新Agentic RAG后，检验白板生成更精准：

  ```bash
  # 生成检验白板
  "@离散数学.canvas 生成检验白板"

  # 新系统会：
  # 1. 从Graphiti获取薄弱概念
  # 2. 从LanceDB获取相关内容
  # 3. 使用Weighted融合 (薄弱点权重70%)
  # 4. Cohere Reranking提升质量
  # 5. 生成针对性检验题
  ```

  ### 4. 艾宾浩斯复习系统

  系统自动追踪学习进度，在最佳时间点提醒复习：

  - **触发点1**: 学习后24小时
  - **触发点2**: 7天后
  - **触发点3**: 30天后
  - **触发点4**: 基于FSRS动态调整

  ## 5. 故障排查

  ### 5.1 Neo4j连接失败

  ```
  症状: ConnectionError: Unable to connect to Neo4j
  原因: Neo4j服务未启动或端口被占用

  解决:
  1. 检查Neo4j容器状态: docker ps | grep neo4j
  2. 检查端口: netstat -an | grep 7687
  3. 重启Neo4j: docker restart neo4j-canvas
  ```

  ### 5.2 检索延迟过高

  ```
  症状: P95 > 400ms
  原因: Neo4j查询慢或网络问题

  解决:
  1. 检查Neo4j索引: SHOW INDEXES
  2. 添加索引: CREATE INDEX FOR (n:Concept) ON (n.name)
  3. 检查网络延迟: ping localhost
  ```

  ### 5.3 Cohere API失败

  ```
  症状: Cohere rerank failed, falling back to local
  原因: API限流或网络问题

  解决:
  1. 检查API Key: echo $COHERE_API_KEY
  2. 检查限流: 查看Cohere dashboard
  3. 降级到Local Reranking (自动)
  ```

  # Agentic RAG API 文档

  ## 概述

  Epic 12的Agentic RAG提供了增强的检索能力。

  ## 核心API

  ### `canvas_agentic_rag.ainvoke()`

  异步调用Agentic RAG检索。

  **请求格式**:
  ```python
  result = await canvas_agentic_rag.ainvoke({
      "query": "什么是逆否命题?",
      "canvas_file": "离散数学.canvas",
      "is_review_mode": False,  # 可选，复习模式
      "fusion_strategy": "rrf",  # 可选: rrf, weighted, cascade
      "reranking_strategy": "local"  # 可选: local, cohere
  })
  ```

  **响应格式**:
  ```python
  {
      "query": "什么是逆否命题?",
      "graphiti_results": [...],  # Graphiti检索结果
      "lancedb_results": [...],   # LanceDB检索结果
      "fused_results": [...],     # 融合后结果
      "reranked_results": [...],  # Reranking后结果
      "quality_score": 0.85,      # 质量分数
      "latency_ms": 320           # 延迟(ms)
  }
  ```

  ### 错误码

  | 错误码 | 描述 | 处理方式 |
  |--------|------|----------|
  | `GRAPHITI_UNAVAILABLE` | Graphiti连接失败 | 降级到LanceDB单源 |
  | `COHERE_TIMEOUT` | Cohere API超时 | 降级到Local Reranking |
  | `QUALITY_LOW` | 检索质量低 | 触发Query重写 |

  ### 使用示例

  ```python
  from src.agentic_rag.state_graph import canvas_agentic_rag

  # 基础检索
  result = await canvas_agentic_rag.ainvoke({
      "query": "逆否命题的定义"
  })

  # 检验白板生成 (高质量)
  result = await canvas_agentic_rag.ainvoke({
      "query": "检索薄弱概念",
      "canvas_file": "离散数学.canvas",
      "fusion_strategy": "weighted",
      "reranking_strategy": "cohere"
  })

  # 复习模式
  result = await canvas_agentic_rag.ainvoke({
      "query": "复习逆否命题",
      "is_review_mode": True
  })

- [ ] 4.2: 创建 `scripts/health_check.py`
  ```python
  #!/usr/bin/env python3
  """健康检查脚本"""
  import sys

  def check_neo4j():
      try:
          from neo4j import GraphDatabase
          import os
          uri = os.environ.get("NEO4J_URI", "bolt://localhost:7687")
          driver = GraphDatabase.driver(uri, auth=("neo4j", "password123"))
          driver.verify_connectivity()
          driver.close()
          return True, f"Connected ({uri})"
      except Exception as e:
          return False, str(e)

  def check_lancedb():
      try:
          import lancedb
          import os
          path = os.environ.get("LANCEDB_PATH", "./data/lancedb")
          db = lancedb.connect(path)
          return True, f"Ready ({path})"
      except Exception as e:
          return False, str(e)

  def check_agentic_rag():
      try:
          from src.agentic_rag.state_graph import canvas_agentic_rag
          # 验证graph已编译
          if canvas_agentic_rag:
              return True, "Compiled"
          return False, "Not compiled"
      except Exception as e:
          return False, str(e)

  def main():
      checks = [
          ("Neo4j", check_neo4j),
          ("LanceDB", check_lancedb),
          ("Agentic RAG", check_agentic_rag),
      ]

      all_passed = True

      for name, check_fn in checks:
          passed, message = check_fn()
          status = "[OK]" if passed else "[FAIL]"
          print(f"{status} {name}: {message}")
          if not passed:
              all_passed = False

      return 0 if all_passed else 1

  if __name__ == "__main__":
      sys.exit(main())
  ```
